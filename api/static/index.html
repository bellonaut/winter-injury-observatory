Urban Winter Injury Risk & Equity Observatory

Portfolio demo: a production-shaped FastAPI service that predicts neighborhood-level winter injury risk using deterministic synthetic data and a bundled local model artifact.

This deployment is tuned for reliability in recruiter review flows: fast startup, clear health reporting, and reproducible inference.

Open API Docs Health Endpoint Service Metadata

Auth

Use bearer token configured via DEMO_API_TOKEN.

Model Backend

Default is local artifact via MODEL_BACKEND=local.

Smoke Check

Expected healthy paths: /health, /docs, /predict.

Example Request

curl -X POST "$BASE_URL/predict" -H "Authorization: Bearer $DEMO_API_TOKEN" -H "Content-Type: application/json" -d '{"temperature":-15.5,"wind_speed":25.0,"wind_chill":-28.0,"precipitation":2.5,"snow_depth":30.0,"hour":8,"day_of_week":1,"month":1,"neighborhood":"Downtown","ses_index":0.45,"infrastructure_quality":0.70}'
